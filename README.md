# Rent-A-Car Web Application

A web-based car rental management system developed using PHP and MySQL. 
The application allows users to browse available cars, make bookings, and manage rental records.

## Features
- User registration and login
- Car availability search
- Booking management
- Admin panel for vehicle management

## Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## How to Run
1. Install XAMPP or WAMP
2. Import the database into MySQL
3. Place the project in the `htdocs` folder
4. Access via `http://localhost/`
